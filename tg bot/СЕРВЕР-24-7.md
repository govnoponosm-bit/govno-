# Бот 24/7 — что могу и что нужно сделать вам

Я **не могу** сам купить сервер и вставить ваш токен: для этого нужны **ваш аккаунт** и **оплата** (или бесплатный тариф).  
Ниже — самый простой способ: **Railway** (без Linux и WinSCP).

---

## Вариант 1 — Railway (проще всего, ~$5/мес или пробный период)

### 1. Регистрация
1. https://railway.app → Sign up (через GitHub удобнее)
2. Подтвердите email

### 2. Проект на GitHub (один раз)

1. https://github.com/new → репозиторий `tg-movie-bot` (Private)
2. На компьютере в PowerShell:

```powershell
cd "C:\Users\anyab\OneDrive\Документы\cursor\tg bot"
git init
git add bot.py config.py database.py keyboards.py middlewares.py states.py requirements.txt Dockerfile .dockerignore railway.toml handlers services .env.example .gitignore README.md
git commit -m "bot"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/tg-movie-bot.git
git push -u origin main
```

(Замените `ВАШ_ЛОГИН`. Файл `.env` в git **не добавляйте** — там токен!)

### 3. Деплой на Railway

1. Railway → **New Project** → **Deploy from GitHub repo**
2. Выберите `tg-movie-bot`
3. Откройте сервис → **Variables** → добавьте:

| Variable | Value |
|----------|--------|
| `BOT_TOKEN` | токен от BotFather |
| `ADMIN_IDS` | ваш ID (только цифры) |

4. **Settings** → **Deploy** — дождитесь зелёного статуса
5. **Settings** → включите **Restart Policy** / always on если есть

### 4. Остановите бота на ПК

В терминале на компьютере: **Ctrl+C**, если `bot.py` ещё запущен.

### 5. Проверка

Telegram → `/start` у бота.

## Вариант 2 — VPS (дешевле в долгую, ~200–400 ₽/мес)

Полная инструкция: файл **`КАК-ЗАГРУЗИТЬ-НА-СЕРВЕР.md`**

Кратко: Timeweb/Aeza → Ubuntu → WinSCP → `/opt/tg-bot` → `./deploy/install.sh`

## Вариант 3 — Docker на любом сервере

Если на VPS установлен Docker:

```bash
cd /opt/tg-bot
# создайте .env с BOT_TOKEN и ADMIN_IDS
docker build -t tg-bot .
docker run -d --name tg-bot --restart always --env-file .env -v $(pwd)/bot.db:/app/bot.db tg-bot
```

## Важно

1. **Один токен** — бот только на сервере **или** только на ПК, не вместе.
2. **ADMIN_IDS** — только цифры: `123456789`
3. База `bot.db` на Railway **может сброситься** при пересборке — для важных данных лучше VPS с постоянным диском.
